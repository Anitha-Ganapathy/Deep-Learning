def stft(data):
  S_stft = librosa.stft(data, n_fft=1024, hop_length=512)
  return S_stft

Pre-process train and test data by creating positive sample of combining spectrograms of same speakers and negative samples of combining spectrograms of different speakers.I have done zero padding for train spectograms to have 45x513 dimensions as the train spectogram is shorter than the test

Create Positive Pairs¶

Generates all combinations of pairs and randomly selects L pairs from it.

In [8]:
def create_pos_pairs(speaker, L=45):
    batch = []
    all_pairs = list(combinations(range(10), 2))
    l_pairs = random.sample(all_pairs, L)
    for a,b in l_pairs:
        stft_1 = np.abs(librosa.stft(speaker[a], n_fft=1024, hop_length=512)).T
        stft_2 = np.abs(librosa.stft(speaker[b], n_fft=1024, hop_length=512)).T
        batch.append([stft_1, stft_2])
    return batch

Create Negative Pairs¶

The Main Speaker's utternaces lie between start and end. The other 49 speakers' utterances like before start and after end. Randomly sample L utterances from the main speaker's utterances with replacement as L is greater than the number of utterances by the main speaker, i.e., L>10. Whereas, we randomly sample L utterances from the other 49 speakers' utterances without replacement as L is less than their total utterances L<490. Using the samples from the above 2 steps we create Negative Pairs

In [9]:
def create_neg_pairs(pos_sp_num, data, L=45):
    batch = []
    
    start = pos_sp_num*10
    end = start + 10     
    pos_spk = data[start: end]
    neg_spk = data[:start] + data[end:]
    neg_sample = random.sample(neg_spk, L)
        
    for l in range(L):
        pos = random.choice(pos_spk)
        
        stft_pos = np.abs(librosa.stft(pos, n_fft=1024, hop_length=512)).T
        stft_neg = np.abs(librosa.stft(neg_sample[l], n_fft=1024, hop_length=512)).T
        batch.append([stft_pos, stft_neg])

    return batch

Siamese network 2 identical CNN 2D network which takes the spectrograms of 2 speakers (same or different) and make do the inner product of the 2 fixed lenght ouput vectors from 2 networks.

The CNN network will have 2 convolution layer, 2 max pooling for each of the convolution layer, and 2 fully connected layer with 1 dropout of 80%

def siamese_network():  
  left_input = Input(shape=(None, 513))
  right_input = Input(shape=(None, 513))
  
  xavier_init = tf.keras.initializers.glorot_normal
  
  # base_model = Sequential([
  #     Conv2D(filters= 20, kernel_size=(4, 2), activation='relu', strides =1, 
  #             kernel_initializer= xavier_init, # padding = 'same',
  #             input_shape= (45, 513, 1)),
  #     MaxPooling2D(pool_size=(2, 2), strides=2, padding='same'),
  #     Conv2D(filters=20, kernel_size=(4, 2), activation='relu', strides=1,
  #           # padding = 'same',
  #            kernel_initializer= xavier_init),
  #     MaxPooling2D(pool_size=(2, 2), strides=2),
  #     Flatten(),
  #     Dense(units=200, activation='relu' , kernel_initializer=xavier_init),
  #     Dropout(0.7),
  #     Dense(units=513, activation='tanh' , kernel_initializer=xavier_init)
  # ])


  base_model = Sequential([
      LSTM(128, return_sequences=True),
      BatchNormalization(),
      Dropout(0.8),
      LSTM(64, return_sequences=False),
      BatchNormalization(),
      Dropout(0.8),
      Dense(64, activation='tanh', kernel_regularizer='l2'),
      Dropout(0.2)
  ])

  left_output = base_model(left_input)
  right_output = base_model(right_input)

  print(K.shape(left_output))
  print(K.shape(right_output))
  
  # prediction = tf.reduce_sum(tf.multiply(left_output, right_output), axis = 1)
  # prediction = tf.reshape(prediction, [-1, 1])
  
  inner_product = Dot(axes=1)([left_output, right_output])
  prediction = sigmoid(inner_product)
    
  siamese_model = Model(inputs = [left_input, right_input], outputs = prediction)

  return siamese_model
